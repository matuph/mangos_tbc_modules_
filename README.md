## Welcome to C(ontinued)-MaNGOS

This is cmangos tbc with modules.
There are two patches in the repo that need to be renamed, simply rename tbc.patch.ini to tbc.patch and tbc-fix.patch.ini to tbc-fix.patch.

execute patch with the following command
git apply tbc.patch
git apply tbc-fix.patch
